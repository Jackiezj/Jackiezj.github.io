013. python面向对象及提高

1. 类和对象

  • 类

    定义: 相同特征,行为的事物

  • 对象

    • 定义: 一个具体事物
    • 组成: 属性+方法
    • 创建对象
      ​ 对象名 = 类名()
    • 调用方法
      ​ 对象名.方法名()
    • 类外定义属性
      ​ 对象.属性 = 值

  • 类和对象的关系

    类是对象的抽象概念, 对象是类的实例

  • self

    表示调用该方法的对象

  • __init__和__str__

  • 私有

    私有属性: __attr
  私有方法: __func()

  • 类属性和实例属性

    • 类属性: 定义在类中的属性
      ​ 只能通过类对象修改, 不能通过实例对象修改
    • 实例对象修改, 实质上是增加一个同名的实例属性
      ​ 实例属性: 定义init中或类外

  • 类方法和静态方法

    类属性和类方法都可以通过对象或类名调用

    方法中不使用实例对象或类对象时,定义静态方法

    类中定义了同名的对象方法、类方法、静态方法时，调用方法会优先执行最后定义的方法

2. 封装

​ 类, 方法, 属性都是一种封装

3. 继承

  • 单继承
    ​ class 子类名(父类名):

  • 多层继承
    ​ C继承B, B继承A,则C有A,B的属性和方法

  • 重写
    实现父类的同名方法

  • 多继承

4. 多态

同类对象的多种形态

  • 实现步骤
    • 继承, 重写
    • 子类对象调用父类处理

5. 单例

  • __new__()
    ​ 自定义对象的创建过程 作用: 返回父类创建的对象

    ​ 区别__init__: init初始化, new创建并返回对象

  • 单例模式

6. 异常

  • 异常处理

  • 异常传递

    • try嵌套时, 如果内层try无法捕获该异常,就会向外层try进行传递
    • 函数嵌套时, 如果内层函数无法捕获该异常, 就会向外层函数进行传递

  • 自定义异常
