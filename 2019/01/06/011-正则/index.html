1.概述

  • 查找验证特定格式的字符串

    应用：匹配数据，提取数据

2.re模块

  • re的基本使用

  • re的其它功能

3.匹配一个字符

符号 作用
. 匹配任意一个
\d 匹配一个数字
\w 匹配一个字符 （a-z,A-Z,0-9,_ ,中文）
\s 匹配空白 （空格, /t, /n）
[ab] 匹配任意一个字符 ( [a-z] [0-9] [12a-z])

\D \W \S 匹配非数字，非单词字符，非空白

4.匹配多个字符

符号 作用
{n} 匹配上个字符n次 eg：1[3-9]\d{9}
{m,n} 匹配上个字符 m 到 n 次 eg: {5, } 5到无限个
? 匹配上个字符 1 或 0 次
* 匹配上个字符 0 或 无限次
+ 匹配上个字符 1 或 无限次

5.匹配开头结尾

符号 作用
^ 匹配开头
$ 匹配结尾

匹配标识符 [a-zA-Z_][a-zA-Z0-9_]*$
a[^>]b [^>]表示非>

6.分组

符号 作用
\ (竖杠) 匹配左右任意一个表达式
(ab) 将()作为一个分组
\4 引用第4个()匹配的字符串
(了解)(?P<name>) 分组起别名
(了解)(?P=name) 引用别名为name分组匹配到的字符串

7.python贪婪和非贪婪

python对正则默认贪婪
非贪婪：在正则成立的条件下，取的越少越好
贪婪 ——> 非贪婪 ： *? +? {m,n}?

8.r的作用

r表示原生字符串，r”” 引号中的内容不会转义
